# 草台编子识字班项目上下文

## 项目定位

草台编子识字班是一个偏平民编程、AI 编程、互助学习和左翼气质的内部课程社区。

它不是一个商业化包装优先的课程站，而是一个面向普通人的数字时代识字班：让学员学会用 AI 和代码完成真实创造，并能通过公开协作参与到项目本身。

## 核心产品

- 公开主页和前言文章。
- 需要卡密解锁的正式课程。
- 需要卡密后才能访问的极简论坛。
- 主页学员表，鼓励学员通过 GitHub PR 把自己加入进去。
- 注册时收集邮箱和 QQ 号；QQ 号用于论坛和主页头像。
- QQ 头像通过短缓存代理引用腾讯头像源，让头像能自动更新。

## 内容与权限原则

公开仓库可以放：

- 前言、招生说明、试看内容。
- 课程目录、标题、摘要和结构。
- 学员表数据。
- 公开设计资产和插画资产。
- 技术方案、协作规则、ADR。

公开仓库不能放：

- 付费课程正文。
- 需要卡密才能访问的论坛正文备份。
- 会泄漏付费内容的前端 JSON、静态 HTML 或构建产物。

付费课程正文和受限论坛内容默认存入 Cloudflare D1，由服务端鉴权后读取和渲染。

## 技术方向

当前倾向：

- Next.js / React
- Fumadocs 或 MDX 用于公开文档体验
- Better Auth 做邮箱登录
- Drizzle ORM
- Cloudflare D1
- Cloudflare Workers / Pages
- 极简自研论坛

## 品牌与美术

Logo 已确定为野草长成电路的简化版本，资产在 `设计资产/logo.svg` 和 `设计资产/logo.png`。

前言插画风格已沉淀为“数字时代的工人版画”或“工业寓言式新木刻”。详见 `docs/草台编子识字班插画美术规范.md`。

## 协作方式

学生 PR 是课程的一部分，不只是代码贡献。Agent 处理外部 PR 时，要把它当成教学现场的一部分：指出问题时要具体、友好、可执行。

## 领域词汇

- **学员通行证**：当前实现里的 `course:full` entitlement。它解锁正式课程、受限论坛、MCP 与外部 API 的学员能力；未来新增更细 scope 时，不得让既有学员掉权限。
- **Agent 访问令牌**：某个用户授权给 MCP / 外部 API 客户端的调用凭证。令牌本身不创造产品权益，实际权限必须同时满足令牌 scope 与用户仍有效的 entitlement。
