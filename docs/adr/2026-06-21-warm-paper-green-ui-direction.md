# 整站 UI 视觉基调：暖纸墨绿

- 日期：2026-06-21
- 状态：已采纳（Accepted）

## Context

`docs/草台编子识字班插画美术规范.md` 沉淀的是**插画**风格——重黑红木刻、工业寓言、巨物压迫感。但那是为文章首图准备的，不能直接当作整站页面框架的底色：

- 整站若全黑红，长篇课程正文与前言的可读性会很差，违背"数字时代识字班"应当好读、好亲近的定位。
- logo 本身用的是另一套语言：米色纸底 + 暗绿墨线 + 红色点缀 + 金/青节点。它代表"人间材料、学习、手工感"那一面。
- 美术规范里也明确写了色彩的分工：黑色负重量，红色负情绪，米色负人味，绿色只做点睛。

因此需要在"插画的重黑红"与"页面的可读暖纸"之间定一条清晰边界，避免每个页面各凭手感、风格漂移，也避免误入通用 SaaS / 玻璃拟态（见 `docs/agents/collaboration.md` 的设计边界）。

## Decision

整站页面框架（背景、排版、导航、卡片、表单）采用 **暖纸墨绿** 基调；**重黑红木刻只作为"精神段落"的插画带与少数深色区块出现**，不蔓延成整页底色。

设计 token（落在 `app/globals.css` 的 `@theme`）：

- 背景纸：`#F7F0DF`；次级纸面 `#F0E6CF` / `#E8DCC0`；暖色描边 `#DDCFA9`
- 正文墨：`#20352B`（暗绿炭黑）；次级 `#44544A`；弱化 `#74786A`
- 强调红：`#B3261E`（情绪/警示/链接 hover）；重红 `#8A1812`
- 点缀：金 `#D7A83F`、青 `#39BFC9`、冷绿 `#3AA46A`（点睛，勿铺满）
- 深色木刻区底：`#161210` / `#241D18`，其上用纸色文字

排版与肌理：

- 标题用黑体（粗黑刻痕感），长文正文用宋体（旧讲义/夜校讲义感）。
- 印刷块用硬投影（无模糊偏移阴影），模拟贴上去的版画；标题可加轻微红色误套色。
- 明确避免：明亮科技蓝紫渐变、玻璃拟态、干净扁平 SaaS 配色、过度赛博霓虹。

场景切换：公开站（首页/前言/招生/学员墙）走暖纸墨绿；未来课程阅读区与论坛可在此基础上引入更暗的"终端感"区块，但仍从这套 token 派生，不另起一套视觉语言。

## Consequences

- 所有页面共享一套 token，新页面有据可依，风格不会一页一个样。
- 插画与页面有清晰分工：插画继续严格遵循美术规范的重黑红木刻，不被"调亮"以迁就页面。
- 代价：深色区块（木刻带、未来论坛/阅读区）需要单独的对比度与可读性校验，不能直接套浅色 token。
- 若日后整站要改走"重黑红印刷"或"双模式"，需以新 ADR 取代本条，并迁移 `@theme` token。
